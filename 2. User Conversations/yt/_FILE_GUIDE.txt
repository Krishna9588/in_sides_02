🎯 WHAT IT DOES
─────────────────────────────────────────────────────────────────────────────

Single Video:
  ✓ Extract title, channel, views, likes, duration
  ✓ Download subtitles (manual & auto-generated)
  ✓ Extract top N comments with likes
  ✓ Get description, thumbnail, upload date

Entire Channel:
  ✓ Scrape top N videos
  ✓ Get all data for each video
  ✓ Channel statistics
  ✓ Success tracking

Output:
  ✓ Structured JSON format
  ✓ Auto-saved with timestamps
  ✓ Easy to parse and analyze
  ✓ Excel/database compatible

📋 COMMON TASKS
─────────────────────────────────────────────────────────────────────────────

Extract single video:
  python youtube_scraper_production.py "URL"

Extract channel (top 5):
  python youtube_scraper_production.py "URL" --channel --count 5

Get more comments:
  python youtube_scraper_production.py "URL" --comments 30

Save to file:
  python youtube_scraper_production.py "URL" --output result.json

Verbose mode (see what's happening):
  python youtube_scraper_production.py "URL" --verbose

Interactive mode:
  python youtube_scraper_production.py --interactive

See help:
  python youtube_scraper_production.py --help
